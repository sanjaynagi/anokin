**METHODOLOGY FOR SEQUENCING MOSQUITO ORDER NO. DMo25-3087 USING DARTSEQTM**  
Mosquito samples were sent to SEQART AFRICA located at International Livestock Research Institute in Nairobi for Genotyping. DNA extraction was done from tissue using Nucleomag Tissue DNA extraction kit. The genomic DNA extracted was in the range of 50-100ng/ul. The genomic DNA quality and quantity were checked on 0.8% agarose. Libraries were constructed according to  ([Kilian et al., 2012](https://www.frontiersin.org/articles/10.3389/fgene.2017.00098/full#B32)) DArTSeq complexity reduction method through digestion of genomic DNA using a combination of  PstI \_HPaII enzymes and ligation of barcoded adapters and common adapter followed by PCR amplification of adapter-ligated fragments. Libraries were sequenced using Single Read sequencing runs for 138 cycles. Next generation sequencing was carried out using NovaseqX.   
SEQART AFRICA uses genotyping by Sequencing (GBS) DArTseqTM technology, which provides rapid, high quality and affordable genome profiling, even from the most complex polyploid genomes. DArTseq markers scoring was achieved using DArTsoft14 which is an in-house marker scoring pipeline based on algorithms. Two types of DArTseq markers were scored, SilicoDArT markers and SNP markers which were both scored as binary for presence /absence (1and 0, respectively) of the restriction fragment with the marker sequence in genomic representation of the sample. Both SilicoDArT markers and SNP markers were aligned to the reference genomes of VectorBase-68\_AfunestusAfunGA1\_Genome, to identify chromosome positions.
